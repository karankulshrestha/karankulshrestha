### Experience

#### Stock GPT — AI-Powered Market Analysis Platform &nbsp; | &nbsp; [Go India Advisors](https://www.goindiastocks.com/GIA)

Built a **state-of-the-art agentic AI assistant** for equity research, serving **top mid-cap Indian stock exchange companies, fund managers, and individual investors**. The platform performs deep research across **broker reports, earnings transcripts, quarterly/annual filings, financial data, and 27+ SQL tables** using multi-tool calling with the latest AI models. Implemented credit systems and monitoring with Langfuse, driving a **50% increase in paid users**.

#### Multi-Node Elasticsearch Vector Database

Deployed a **multi-node Elasticsearch cluster from scratch** as a scalable vector database with proper data backups and recovery — replacing Pinecone to optimize costs at scale.

#### GPU-Powered Intelligent Indexers

Built multiple **GPU-powered indexing pipelines**, including one using **MinerU** to extract and crop images from raw broker reports, then analyze them in context using **Gemini** — achieving **95% extraction accuracy**. Dynamically scaled GPU instances for task-specific processing, enabling analysts to search and integrate visual data across Indian stock companies while significantly reducing infrastructure costs.

#### Python Backend & Automation

Extensive Python scripting for **schedulers, data migration pipelines, and backend APIs** powering production financial systems.
